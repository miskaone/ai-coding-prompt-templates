# Domain-Specific Templates

Customized templates for specific industries and use cases.

## Available Domains

- **[FinTech](fintech.md)** - PCI-DSS compliance, payment processing
- **[HealthTech](healthcare.md)** - HIPAA compliance, PHI handling
- **[SaaS](saas.md)** - Multi-tenancy, authentication, RBAC

- Supply Chain (inventory, logistics)
- Education (student data, FERPA)
