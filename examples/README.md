# Examples

This directory contains language-specific examples demonstrating template usage.

## Structure

```
examples/
├── python/          # Python examples
├── typescript/      # TypeScript examples
├── go/              # Go examples
└── rust/            # Rust examples
```

## Contributing Examples

We welcome examples in any language! See [CONTRIBUTING.md](../CONTRIBUTING.md) for guidelines.

**What makes a good example:**
- Shows real security/performance issue that template caught
- Includes before/after comparison
- Explains what reflection revealed
- Is practical and realistic (not contrived)

**Example format:**
```markdown
# [Template Name] Example ([Language])

## Scenario
[Real use case]

## Template Usage
[How you filled in the template]

## AI Output
[Generated code]

## What Reflection Caught
- [Issue 1]
- [Issue 2]
```
